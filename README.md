# Задачи ТЮФ / ТЮЕ

Веб-приложение для учёта идей и задач турниров **ТЮФ**, **ТЮЕ** и **конкурса капитанов**.  
Вместо поиска по Telegram-чату — общая база со статусами, канбаном, комментариями и выгрузкой.

## Возможности

- **Карточки задач** — номер идеи (или «Нет номера идеи»), условие, автор, ссылки, дата сообщения в Telegram
- **Назначения** — ТЮЕ · ТЮФ и ТЮЕ · Конкурс капитанов
- **Канбан** — три отдельные доски; перетаскивание меняет статус
- **Формулировки** — черновик из чата → формулировка перед отправкой → итоговая (когда задача в турнире)
- **Турнир** — при статусе «Играется»: ТЮФ/ТЮЕ, год, номер задачи (для КК — полуфинал/финал и год)
- **Комментарии** — несколько авторов; можно добавить при создании или позже
- **Выгрузка** — TXT и CSV (с комментариями)
- **Доступ** — три логина (Никита, Артём, Илья), смена пароля в настройках

## Статусы

1. Сообщение в Telegram  
2. Формулировка перед отправкой  
3. Отправлена в методкомиссию *(только доска «ТЮФ и ТЮЕ»)*  
4. Играется в турнире  
5. Отклонена  

При перетаскивании в «Формулировку» или «Играется» открывается форма — нужно заполнить соответствующие поля.

## Обязательные поля при создании

- Условие  
- Автор  
- Назначение  
- Дата и время в Telegram  

## Запуск

```bash
cp .env.example .env   # задайте пароли
chmod +x run.sh
./run.sh
```

Откройте http://127.0.0.1:8000

| Логин | Кто | Пароль по умолчанию |
|-------|-----|---------------------|
| `nikita` | Никита | `change-me` |
| `artem` | Артём | `change-me` |
| `ilya` | Илья | `change-me` |

После первого входа смените пароль: **Настройки**.

### Зависимости

Python 3.10+, затем:

```bash
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt
```

(`./run.sh` создаёт venv сам, если его ещё нет.)

## Данные

- База: `data/zadachi.db` (лежит в репозитории)
- Пароли: `.env` (в git **не** попадает — у каждого свой файл на сервере/компьютере)

После локальных правок задач сделайте commit + push, чтобы база обновилась у остальных.
