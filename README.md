# Задачи ТЮФ / ТЮЕ

Веб-приложение для совместной работы над задачами научных турниров **ТЮФ**, **ТЮЕ** и **конкурса капитанов**.

Идеи раньше копились в Telegram-чате и раз в год вручную выбирались на турнир. Этот сервис заменяет хаос чата структурированной базой: статусы, канбан-доски, формулировки, комментарии и выгрузка.

## Возможности

- Карточки задач с условием, автором, источниками и датой сообщения в Telegram
- Три доски по назначению: **ТЮЕ**, **ТЮФ и ТЮЕ**, **конкурс капитанов**
- Канбан со статусами жизненного цикла задачи
- Отдельные поля формулировок: черновик → перед отправкой в методкомиссию → итоговая в турнире
- Метаданные участия: турнир, год, номер задачи (для КК — этап и год)
- Комментарии от нескольких авторов
- Экспорт в TXT и CSV
- Простая авторизация для небольшой команды

## Жизненный цикл задачи

1. Сообщение в Telegram  
2. Формулировка перед отправкой  
3. Отправлена в методкомиссию *(для доски «ТЮФ и ТЮЕ»)*  
4. Играется в турнире  
5. Отклонена  

Перетаскивание карточки на канбане меняет статус; при переходе к формулировке или «играется» открывается форма с нужными полями.

## Стек

Python · FastAPI · SQLite · Jinja2 · SortableJS

## Запуск

```bash
cp .env.example .env
chmod +x run.sh
./run.sh
```

Приложение: http://127.0.0.1:8000  

Учётные данные задаются в `.env` (файл в репозиторий не входит).

Требования: Python 3.10+.

## Структура

```
app/           — backend и веб-интерфейс
data/          — SQLite-база задач
.env.example   — шаблон конфигурации
run.sh         — локальный запуск
```
