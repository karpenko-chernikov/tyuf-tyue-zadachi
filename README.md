# ТЮФ / ТЮЕ — задачи

Веб-приложение для учёта и подготовки задач научных турниров **ТЮФ**, **ТЮЕ** и **конкурса капитанов**.

Команда ведёт идеи от сообщения в Telegram до формулировки и участия в турнире: статусы, канбан, комментарии, вложения, история правок, импорт из экспорта Telegram Desktop и выгрузка в TXT/CSV.

## Стек

Python 3.10+ · FastAPI · SQLAlchemy · SQLite · Jinja2 · SortableJS

## Возможности

- Карточки задач: условие, автор, источники, дата в Telegram
- Три доски по назначению и канбан по статусам жизненного цикла
- Формулировки на разных этапах (черновик → методкомиссия → итоговая)
- Метаданные турнира: год, номер задачи, этап конкурса капитанов
- Комментарии и файлы (в том числе фото и видео) к задаче и к комментариям
- Импорт локального JSON-экспорта Telegram с разбором идей и медиа
- История изменений, сессионный вход, экспорт по фильтру

## Запуск локально

```bash
cp .env.example .env   # задайте SECRET_KEY
chmod +x run.sh
./run.sh
```

Откройте http://127.0.0.1:8000

Файл базы `data/zadachi.db` в репозиторий не входит: при первом запуске создаётся пустая БД. Начальные пользователи и пароли задаются переменными `PASS_*` в `.env` один раз; дальше пароли меняются в настройках приложения.

## Docker

```bash
docker build -t tyuf-tyue-zadachi .
docker run -p 8000:8000 \
  -e SECRET_KEY='замените-на-случайную-строку' \
  -v "$(pwd)/data:/app/data" \
  tyuf-tyue-zadachi
```

Том `data` нужен, чтобы SQLite и вложения сохранялись между перезапусками.

## Структура

```
app/            код приложения
data/           SQLite и импорты (локально, не в git)

```
