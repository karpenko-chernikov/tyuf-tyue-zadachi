# ТЮФ / ТЮЕ — задачи

Сервис учёта и подготовки задач для научных турниров школьников: **ТЮФ**, **ТЮЕ** и **конкурс капитанов**.

Идеи появляются в рабочем Telegram-чате, а отбор и доработка идут в течение сезона. Приложение держит единый реестр задач, статусы подготовки и артефакты (формулировки, источники, файлы), чтобы команда могла совместно вести портфель до публикации списка турнира.

## Предметная область

Каждая задача привязана к назначению (только ТЮЕ / ТЮФ и ТЮЕ / конкурс капитанов) и проходит жизненный цикл:

1. сообщение в Telegram  
2. формулировка перед отправкой  
3. рассмотрение методкомиссией (для доски ТЮФ и ТЮЕ)  
4. играется в турнире либо отклонена  

Для участвующих задач хранятся турнир, год, номер в списке; для конкурса капитанов — этап. Поддерживается архив задач, которые больше не предлагаются.

## Архитектура

Классическое server-rendered веб-приложение на FastAPI:

- маршруты и формы — FastAPI + Jinja2  
- состояние — SQLAlchemy ORM, по умолчанию SQLite  
- сессии — cookie-based (`SessionMiddleware`)  
- вложения — BLOB в той же БД (изображения и видео отдаются inline, с поддержкой HTTP Range)  
- канбан — SortableJS, смена статуса через API  
- импорт — разбор JSON-экспорта Telegram Desktop (идеи, комментарии, медиа, привязка к задачам)  
- аудит — журнал изменений полей задачи  

База и медиа не входят в git: на сервере монтируется том `data/`.

## Стек

| Слой | Технологии |
|------|------------|
| Backend | Python 3.10+, FastAPI, Uvicorn, SQLAlchemy 2 |
| UI | Jinja2, CSS, SortableJS |
| Хранение | SQLite (опционально другой URL через `DATABASE_URL`) |
| Конфиг | `python-dotenv`, переменные окружения |
| Доставка | Docker (`python:3.12-slim`) |

## Запуск

```bash
cp .env.example .env
./run.sh
```

Через Docker:

```bash
docker build -t tyuf-tyue-zadachi .
docker run -p 8000:8000 \
  -e SECRET_KEY=… \
  -v "$(pwd)/data:/app/data" \
  tyuf-tyue-zadachi
```

`SECRET_KEY` используется для подписи сессий. Том `data` сохраняет SQLite и вложения между перезапусками.

```
.env.example
```
