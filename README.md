# Задачи ТЮФ / ТЮЕ

Локальная база задач для турниров: идеи из Telegram, статусы, комментарии, выгрузка.

## Быстрый старт

```bash
cd ~/repos/tyuf-tyue-zadachi
chmod +x run.sh
./run.sh
```

Откройте в браузере: **http://127.0.0.1:8000**

При первом запуске создаётся файл `.env` — **обязательно смените пароли** (каждый свой в **Настройках**).

## Вход

| Логин | Имя в системе | Пароль по умолчанию |
|-------|---------------|---------------------|
| `nikita` | Никита | `change-me` |
| `artem` | Артём | `change-me` |
| `ilya` | Илья | `change-me` |

**Смена пароля:** после входа откройте **Настройки** (в шапке) → «Сменить пароль».

## Как пользоваться

1. **Добавить задачу** — кнопка «Добавить» или вставка текста из Telegram.
2. **Автор** — любое имя (не только вы трое); подсказки из уже сохранённых.
3. **Ссылки и источники** — статьи, видео, сайты: по одной ссылке на строку.
4. **Дата и время в Telegram** — обязательное поле.
5. **Комментарии** — можно добавить сразу при создании (с автором) или позже на странице задачи.
6. **Выгрузка** — TXT или CSV из меню.

Для конкурса капитанов варианты ответа пишите прямо в **условие**.

## Назначения

- **ТЮЕ** — только ТЮЕ (биология, химия)
- **ТЮФ и ТЮЕ** — оба турнира (физика / общее)
- **Конкурс капитанов** — задания на КК (варианты ответа, полуфинал/финал)

Отдельное назначение «только ТЮФ» не используется.

## Канбан

В меню **Канбан** — три доски (вкладки). Карточки перетаскиваются между колонками статусов.

При перетаскивании в **«Формулировка»** откроется редактирование — заполните **«Формулировку перед отправлением»**.  
При перетаскивании в **«Играется»** — **«Итоговую формулировку»** и турнир (ТЮФ/ТЮЕ, год, номер задачи; для КК — полуфинал/финал и год).
