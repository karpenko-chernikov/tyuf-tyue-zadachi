2. Проверка перед сохранением

Как работать

  • Лишнее сообщение → тип Пропустить (галочку «Сохранить» ставить не нужно — она снимется сама).
  • Нужная идея → тип Идея + галочка Сохранить.
  • Комментарий → тип Комментарий, в поле К идее выберите задачу (новую из этого списка или уже из базы), галочка Сохранить.
  • Можно сохранить только часть строк, выключить ноут, потом снова нажать «Разобрать»: уже сохранённые по дате Telegram сами станут «пропустить».
